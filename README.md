## **Initial setting**
### **Create a config folder**


<div align="left">

```
                               > config.js

                                 module.exports = {
                                   server_port : 3000,
                                   db_url : 'mariadb://127.0.0.1:3306/local',
                                   db_host : '127.0.0.1', (or localhost)
                                   db_pw : input your password,
                                   db_user : input your user,
                                   db_port : 3306,
                                   db_name : input your DB_NAME
                                 }

                               > email.js { nodemailer.createTransport }

                               > .env     { Jwt Secret Key }

```
</div>
